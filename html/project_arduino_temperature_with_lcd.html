Датчик температуры на Arduino Nano + LCD Экран

Arduino Nano: ATmega328P (OldBootloader)

Display 16x2 LCD in 4-bit mode

Герметичный датчик температуры DS18B20


Собранное


DS18B20 – высокоточный цифровой датчик температуры. Основные характеристики:

Диапазон: -55.. 125 °C

Точность: 0.5 °C

Разрешение: 9.. 12 бит (0.48.. 0.06 °C)

Питание: 3-5.5V

https://arduinomaster.ru/datchiki-arduino/arduino-ds18b20/

Подключение датчика температуры к Arduino

/*
 LiquidCrystal Library - Hello World

 Demonstrates the use a 16x2 LCD display.  The LiquidCrystal
 library works with all LCD displays that are compatible with the
 Hitachi HD44780 driver. There are many of them out there, and you
 can usually tell them by the 16-pin interface.

 This sketch prints "Hello World!" to the LCD
 and shows the time.

 The circuit:
 * LCD RS pin to digital pin 12
 * LCD Enable pin to digital pin 11
 * LCD D4 pin to digital pin 5
 * LCD D5 pin to digital pin 4
 * LCD D6 pin to digital pin 3
 * LCD D7 pin to digital pin 2
 * LCD R/W pin to ground
 * LCD VSS pin to ground
 * LCD VCC pin to 5V
 * 10K resistor:
 * ends to +5V and ground
 * wiper to LCD VO pin (pin 3)

 Library originally added 18 Apr 2008
 by David A. Mellis
 library modified 5 Jul 2009
 by Limor Fried (http://www.ladyada.net)
 example added 9 Jul 2009
 by Tom Igoe
 modified 22 Nov 2010
 by Tom Igoe
 modified 7 Nov 2016
 by Arturo Guadalupi

 This example code is in the public domain.

 http://www.arduino.cc/en/Tutorial/LiquidCrystalHelloWorld

 Датчик температуры пример кода здесь:
 https://arduinomaster.ru/datchiki-arduino/arduino-ds18b20/

*/

// include the library code:
#include <LiquidCrystal.h>
#include <OneWire.h>
#include <DallasTemperature.h>

OneWire oneWire(8);  // порт подключения датчика (D8)
DallasTemperature ds(&oneWire);

// initialize the library by associating any needed LCD interface pin
// with the arduino pin number it is connected to
const int rs = 12, en = 11, d4 = 5, d5 = 4, d6 = 3, d7 = 2;
LiquidCrystal lcd(rs, en, d4, d5, d6, d7);

void setup() {
    // set up the LCD's number of columns and rows:
    lcd.begin(16, 2);
    // Print a message to the LCD.
    lcd.print("Temperature");
    ds.begin();                 // инициализация датчика ds18b20
}

void loop() {
    // set the cursor to column 0, line 1
    // (note: line 1 is the second row, since counting begins with 0):
    delay(1000); // Микросхема измеряет температуру, а мы ждем.
    ds.requestTemperatures();                       // считываем температуру с датчика
    lcd.setCursor(0, 1);
    lcd.print(ds.getTempCByIndex(0));
}
