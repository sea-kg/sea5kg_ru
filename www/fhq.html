Free-Hack-Quest

This is an open source platform for competitions in computer security.

Это движок для проведения игр по информационной безопасности (jeopardy)
а также использование его как архив игр с задачами для тренировки молодого поколения.

образ виртуальной машины можно скачать здесь (Virtual Machine *.ova)

Web-site: http://fhq.sea-kg.com or http://fhq.keva.su

Source code: https://github.com/sea-kg/fhq

Previous platfrom: https://github.com/sea-kg/fhq-2013

Previous platfrom: https://github.com/sea-kg/fhq-2012
